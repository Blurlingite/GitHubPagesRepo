Pichu's Vending Machine

For this project, I was challenged to create a vending machine web application without using any javascript. You can select an item by clicking on one of the blue item buttons, which are dynamically made. Each blue item button tells you the name, price, and inventory of the item. All of this comes from my DaoInMemImpl in my Dao package. Once I get all the information I need to make each item button, I just use JSTL to display all the item buttons with a tag. Although they look like buttons, they are actually links styled to look like buttons.

If you buy an item, and that item’s cost is less than what you put in and it is not out of stock, the item will vend and will display a “thank you” message. If the item cost is more than what you insert, then the application will tell you how much more money you need to insert to buy the item and will let you put in more money. If the item is sold out, not only will it display a “sold out” message but an image of a very sad Pichu will be displayed.
